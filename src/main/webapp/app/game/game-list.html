1.所有小米系列赛事一律禁止任何不公平行为，以下行为包括不限于：修改游戏内部内容（游戏界面、音效等）、第三方程序（辅助软件、外挂）、使用辅助设备（任何形式的外设）、手游PC端模拟器等工具、代打、窥屏、串通比赛，一经发现，取消本次比赛资格和一切已得奖励，视情节严重者，组委会有权做永久禁赛处罚。

2.报名队伍队长需要先加入指定的比赛群进行报名，进群验证须填写：比赛名称+队伍名称。

3.小米超神高校挑战赛，每队须由1名队长，4名队员组成，一支队伍最多5人。参赛队伍一旦组成，则队长或队员在比赛期间不可退出该小队，人员也不可调换，否则赛事组委会有权取消队伍参赛资格。

4.已经报名成功的玩家不可重复参赛，如果赛事组委会发现有重复参赛的情况，赛事组委会有权取消比赛资格及所有已获得的奖励。

5.小米超神高校挑战赛报名队伍数量最高为256支队伍，以报名时间截止为准。

6.小米超神高校挑战赛分为两个阶段，由小组海选赛和晋级赛组成。

7.小米超神高校挑战海选赛：
（1）参加小米超神高校挑战赛队伍海选阶段的选手必须为高校学生并通过赛事官网成功报名。
（2）小组海选赛的比赛时间为4月17日起的每个周末（节假日休息）。
（3）海选形式：以256支队伍为例，参赛队伍分为16组进行BO1比赛，小组决赛为BO3。
（4）晋级赛名额选取方式：各个小组赛冠军进入晋级赛

8.小米超神高校挑战赛晋级赛：
（1）参加小米超神高校挑战赛晋级赛阶段的队伍为海选赛阶段晋级的16支队伍。
（2）晋级赛阶段的晋级方式为单败淘汰赛，赛制BO3，16队伍进行两两对战，最终产生冠亚季军。
（3）晋级赛阶段的季军赛及决赛，赛制为BO5。

9.本次比赛的参赛选手使用自己的比赛账号参加比赛。如发现队伍内有替赛情况，赛事组委会有权判取消队伍参赛资格并取消一切已得奖励。

10.赛前选手有责任调试好比赛用机，确保手机在比赛过程中正常使用。

11.参赛选手需在比赛开始前5分钟内或主办方规定时间内完成调试设备，违反时间将给予警告，情节严重者将剥夺比赛资格。
